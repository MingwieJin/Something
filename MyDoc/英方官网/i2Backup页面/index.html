产品概述
i2Backup 功能强大的企业级数据备份管理软件，为 Windows 和 Linux 混合环境提供完整的数据保护机制，具有保护企业中从工作组到企业级服务器的数据备份能力。管理员能够通过图形界面管理备份和恢复的所有操作，制定企业统一的备份策略。同时针对 Oracle、SQL Server 数据库等提供了备份和恢复解决方案。
产品特点
支持多种恢复方式
支持等多种操作系统
可扩展的结构
简单化操作
应用场景
数据库定时备份
文件级定时备份
块设备定时备份
兼容性列表
Windows：Windows Server2003/2008 标准版 ／ 企业版 ／ 数据中心版 （32位 ／ 64位）／ Windows2012 ／Windows 10 /Windows 2016
Linux： Redhat/Cent OS 4、5、6、7 使用 ext2、ext3、ext4、xfs 等文件系统，SUSE 10、11
