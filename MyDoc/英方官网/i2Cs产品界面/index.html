产品概述
i2CS 基于共享存储的高可用集群软件。实时监控应用系统的运行状态，侦测到故障时执行高可用切换，实现应用级业务接管， 保障业务连续性。
产品特点
灵活的高可用应用模式
全国的国产化兼容性
应用的快速无缝切换
图形化管理
独立于软硬件平台
简单灵活的部署方式
支持MSCS应用
保证集群系统应用
自主研发
满足国产化趋势
应用场景
共享存储集群容灾
远程异地容灾高可用
虚拟化平台集群高可用
兼容性列表
Windows：Windows Server 2003 R2 及以上系统，包括 Windows Server 2008、Windows XP、Windows10 等；
Linux： CentOS 4.4 及以上系统，包括 CentOS 6.7、CentOS 6.5 等；Redhat 6.5 等；SUSE10 sp2 等；
其他：中标麒麟 6.5；Ubuntu12.04.02 等。
