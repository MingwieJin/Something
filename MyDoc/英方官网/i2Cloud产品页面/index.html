产品概述
i2Cloud 私有云灾备解决方案。融合灾备软件和云计算平台，提供开放计费界面，集中管理租户的云灾备资源和服务。适合运营商级别的客户使用，提供远程灾备服务，租户可将数据、应用统一灾备到云平台，数据彼此隔离、安全管理，实现对数据和应用的最大可用性保证。
产品特点
支持多种云存储
字节级增量数据捕获
图形化管理
全面支持各类应用
高效且安全
兼容性列表
Windows：Windows Server 2003R2、Windows Server 2008 及以上系统，Windows XP、Windows10 等；
Linux：CentOS 4.4 及以上系统，包括 CentOS 6.5、7.0、CentOS 7.1 等；Redhat 4.8 及以上系统，SUSE10sp1 及以上系统；
其他：中标麒麟 6.5、中标麒麟 7.2 等；各类虚拟化平台 , Hyper-V，Citrix XenServer，VMware，KVM。
