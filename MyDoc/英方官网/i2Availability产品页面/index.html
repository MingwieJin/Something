产品概述
i2Availability 应用级的高可用保护软件。数据层，实时捕获生产数据并复制到灾备服务器 ; 应用层，实时监控运行状态，若出现异常 ( 如服务异常停止、网络异常、硬件故障、系统宕机 ) 导致业务不可达时，将应用切换到灾备服务器上，通过应用接管实现业务的连续。
产品特点
应用的快速无缝切换
全图形化界面管理
独立于软硬件平台
一对一、多对一
n对n物理机或虚拟技术
安全且可靠
应用场景
异构虚拟化平台之间的应用级高可用
本地机房和云主机间的应用级高可用
任意公有云或私有云之间的应用级高可用
异地机房间、基于专线的应用级高可用
兼容性列表
操作系统：兼容 SUSE Linux 、Redhat、Ubuntu 和 Windows，包括但不限于 SUSE Linux 11 或 Windows Server 2012，并持续支持相应更新版本；兼容中标麒麟，红旗 , FreeBSD 等；
应用软件： 支持但不限于 Oracle、DB2、Cache、MySQL、Informix、SQL_Server、Exchange、Lotus Notes、Sybase ASE 等；
虚拟化平台：支持但不限于 HyperV、VMware、Citrix、OpenStack 等，及国产虚拟化平台如华为 Fusion-Shpere、华三、联想、曙光、浪潮等；
公有云平台：支持但不限于阿里云、腾讯云、盛大云、百度云、AWS、Azure、华为云、UCloud、QingCloud 等云平台。
