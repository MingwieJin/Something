Overview

Enable video option:

video:true

要在carousel中添加video只需要加入<a class="owl-video" href="_straight URL from youtube or vimeo"></a>。

标签不是强制性的，你可以使用其它的标签，但是"owl-video"和url链接中的href属性是必须的。

Additional video options:

videoWidth: false, // Default false; Type: Boolean/Number
videoHeight: false, // Default false; Type: Boolean/Number

Setup

    $('.owl-carousel').owlCarousel({
        items:1,
        merge:true,
        loop:true,
        margin:10,
        video:true,
        lazyLoad:true,
        center:true,
        responsive:{
            480:{
                items:2
            },
            600:{
                items:4
            }
        }
    })
