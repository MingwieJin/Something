Overview

要使用该属性设置autoHeight: true。在同一时间里只有一个项会显示在屏幕上。该属性会计算所有的可见项，并改变它的高度为最高项的高度。

//default settings:
AutoHeight.Defaults = {
    autoHeight: false,
    autoHeightClass: 'owl-height'
};

Setup

$('.owl-carousel').owlCarousel({
    items:1,
    margin:10,
    autoHeight:true
});
