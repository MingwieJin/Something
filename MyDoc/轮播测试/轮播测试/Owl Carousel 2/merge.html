Overview

Merge参数需要在任何子元素（也可以是嵌套结构）上设置data-merge="number_items_to_merge"。这里有一个相似的选项叫mergeFit，该选项可以使merged元素伤害屏幕的大小。

See item 6 on breakpoint below and above 1000px screen width.

Setup

$('.owl-carousel').owlCarousel({
    items:5,
    loop:true,
    margin:10,
    merge:true,
    responsive:{
        678:{
            mergeFit:true
        },
        1000:{
            mergeFit:false
        }
    }
});

html
