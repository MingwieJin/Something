Overview

Stage padding选项可以在stage-wrapper上添加左边和右边的padding(单位像素)。

Option:

stagePadding: Number

Setup

$('.owl-carousel').owlCarousel({
    stagePadding: 50,
    loop:true,
    margin:10,
    nav:true,
    responsive:{
        0:{
            items:1
        },
        600:{
            items:3
        },
        1000:{
            items:5
        }
    }
})
